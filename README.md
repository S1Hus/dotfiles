# My dotfile

This directory contains the dotfiles for my system.

## pre-requesit

Install git and stow
```
apt install git
apt install stow
```

Nvim requires Nerd fonts to be installed 

## Installation

git checkout this repo into your $Home directory.

```
$ git clone git@github.com/...
$ cd dotfiles
```
run GNU stow
```
$ stow .
```
